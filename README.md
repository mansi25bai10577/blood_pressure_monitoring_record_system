Blood Pressure Monitoring System

>Overview of the Project

A Python command-line application designed to help users record, categorize, and analyze blood pressure readings, providing instant feedback and basic statistics to support daily health tracking.

>Features

Add and store systolic/diastolic readings with dates

Automatic classification using AHA guidelines

Stats: average, minimum, maximum readings

Clear text menu for easy navigation

All data is stored locally for privacy

>Technologies/Tools Used

Python 3.x

Standard libraries: datetime, sys

Any text editor (e.g., VS Code, Sublime)

>Steps to Install & Run the Project

Ensure Python 3.x is installed on your system.

Download or clone the project repository.

Open a terminal/command prompt in the project folder.

Run the following command:

text       _
python bp_monitor.py

Follow the menu prompts to use the features.

>Instructions for Testing

Unit Test: Verify each function for correct categorization and statistics.

Integration Test: Simulate typical user scenarios (adding, viewing, analyzing).

Boundary Test: Try extreme values (very high/low readings) and check responses.

Manual Test: Interact with the menu and ensure all features work as expected.
